# HBnB: Project Overview & Design

Welcome to the **HBnB** project. This repository contains the implementation of a custom vacation rental application, designed using a three-layer architecture (Presentation, Business Logic, and Persistence).

---

## Project Overview
HBnB is a simplified clone of AirBnB. The application allows:
* **Users** to register as Clients, Owners, or both.
* **Owners** to list and manage properties (Places) with specific amenities.
* **Clients** to browse listings and submit reviews with ratings (1–5).

---
